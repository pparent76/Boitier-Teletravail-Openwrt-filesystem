Aide

Le mode hors-ligne redirige automatiquement le navigateur vers cette interface d'administration, et ne permet d'accéder à rien d'autre.

Le mode réseau local permet d'accéder au réseau local sur lequel est branché le boîtier,ce qui est principalement utile lorsqu'un réseau public requière une identification via un portail captif pour accéder à Internet.

Le mode entreprise permet de se connecter sur le réseau de l'entreprise distante, correspondant au code de connexion configuré.


Attention

Vous êtes connecté au réseau de l'entreprise uniquement grâce au système de secourt. La connexion au réseau de l'entreprise risque d'être particulièrement lente. La connexion internet utilisée est peut-être trop filtrée, ou bien le serveur Telework-box dans votre entreprise est peut-être mal configuré.
